# Wut iz this?

This is the code that runs http://starwars-countdown.com

# How can I run it?
```
$ cd $GOPATH/src
$ git clone github.com/thrawn01/starwars-countdown.git
$ cd starwars-countdown
$ go build
```

Now run ``./starwars-countdown``
